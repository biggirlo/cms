# cms

html:后台管理页面,包括整体插件包<br />
officialwebsitecService：后台管理端<br/>

## 结构
### 前端：
  html <br />
  >-assets ： 各类插件包 <br />
  >-page <br/>
   >>-common ： 公共配置<br/>
   >>-error :错误页面 <br />
   >>-officialWebsite ：cms页面 <br/>
   >>-system : 系统管理页面 <br />
### 后台 ：
  officialwebsitecService/src/main/java/com/biggirlo/ <br/>
  >-base : 基础核心类与基础工具包<br/>
  >-system ： 系统管理包<br/>
  >-gw : 信息发布包<br/>

## 如何启动：
系统使用springboot，故可使用内置tomcat插件。
